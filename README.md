# Supabase Auth App

A web application built with React that uses **Supabase** for authentication
and backend services.

---

## Tech Stack
- React
- TypeScript
- Supabase (Auth & Database)
- Tailwind CSS

---

## Prerequisites
Make sure you have the following installed:

- Node.js (v18 or higher)
- npm / yarn / pnpm
- Git

## Installation

1. Clone the repository

```bash
git clone https://github.com/ndickers/flowvahub.git
cd flowvahub

npm install

cp .env.example .env
```

VITE_SUPABASE_URL=
VITE_SUPABASE_PUBLISHABLE_KEY=
VITE_APP_URL=

```bash
npm run dev
```
